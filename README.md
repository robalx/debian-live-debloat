# debian-live-debloat
Debian Live 13+ system debloat scripts.
Most comments made in Polish. Use any translator if needed. This set is not perfect - just saves abt 1GB+ (of 4GB+) live ISO image size ( https://cdimage.debian.org/debian-cd/current-live/amd64/iso-hybrid/). Tried on GNOME only.

scripts made to minify Debian Live ISO. Designed for european users (only pl and en locales left). Most non-latine fonts stripped.
